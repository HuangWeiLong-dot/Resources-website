# Resources Hub（本地设计资源导航）

一个轻量的本地静态站点，用来整理常用设计 / 前端资源，并提供统一的视觉风格和可调主题色。整体风格参考现代设计作品集与资源站，强调卡片化、留白和柔和的阴影。

## 核心特性

- **资源分类导航**
  - 首页以大卡片形式展示 8 大资源类别：字体、图片与视频、插画、图标、头像与假数据、灵感、配色、设计与开发工具、实用工具等。
  - 每个类别都有一套定制的 SVG 图标，统一视觉语言，同时与类别语义对应。

- **Collection 资源页**
  - 每个分类对应一个 collection 页面，顶部是带渐变背景的 Hero 卡片，用于展示类别名称与简短描述。
  - 下方为按卡片列表呈现的具体资源，每项包含名称、简洁说明与标签（例如 `Free`、`Commercial`、`API` 等）。
  - 卡片右下角的三角色块在视觉上模拟「折角」，点击可快速打开对应站点。

- **交互与动效**
  - 列表卡片在 hover 时会略微抬起，并出现随鼠标移动的「光线」高亮，增强层次感与可交互性。
  - 首页与 collection 页的内容进入时采用渐入 + 轻微位移动画，从左到右 / 从上到下依次出现。

- **可调主题色**
  - 右上角内置一个小型主题面板，包括当前主色预览和色相滑杆。
  - 通过修改 CSS 变量（`--accent`、`--accent-soft`、`--surface` 等），实现按钮、图标高亮、标签、小角标以及大卡片背景等元素的整体换色。
  - 主题选择会记录在本地存储，方便在不同会话中保持个人偏好。
